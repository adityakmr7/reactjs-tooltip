## Installation

```
  npm install rjs-tooltip
```

or

```
  yarn add rjs-tooltip
```

## Usage

### Using NPM

1. Require rjs-tooltip after installation

```
  import RJSTooltip from 'rjs-tooltip';
```

2. Wrap Your own component

```
  <RJSTooltip>
  	<h1>Hover over me</h1>
  </RJSTooltip>
```
